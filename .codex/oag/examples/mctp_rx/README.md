# MCTP RX OAG Examples

This directory holds IP-specific wavefront examples captured from MCTP RX
closure work. They are not part of the core reusable OAG wavefront template set.

Use these files as concrete references for shaping a project-local wavefront.
For new IP work, start from the generic templates under
`.codex/oag/wavefront-templates/` and copy only the relevant structure into the
target IP's own run plan.
